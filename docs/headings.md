# Heading rules

## Purpose

Headings help users scan content and understand page structure.

## Rules

- Use sentence-style capitalization.
- Start task headings with a verb.
- Keep headings short.
- Do not use punctuation unless the heading is a question.
- Do not skip heading levels.

## Preferred

```md
# Manage projects

## Create a project

## Rename a project

## Delete a project
