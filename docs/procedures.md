# Procedure rules

## Purpose

Procedures help users complete tasks.

## Rules

- Use numbered steps when order matters.
- Use bullets when order does not matter.
- Start each step with an action verb.
- Keep one main action per step.
- Put results after the step, not before it.
- Include expected outcomes when helpful.

## Preferred

```md
1. Open the dashboard.
2. Select **Projects**.
3. Select **Create project**.
4. Enter a project name.
5. Select **Save**.

The new project appears in the project list.
