# Terminology list

Use these terms consistently.

| Preferred term | Avoid | Notes |
|---|---|---|
| select | click, tap | Use for buttons, menu items, checkboxes, and links. |
| enter | type in, input | Use for text fields. |
| sign in | log in, login as a verb | Use “sign in” as the verb. |
| setup | set up | Use “setup” as a noun or adjective. Use “set up” as a verb. |
| API key | api key, API token | Use only when referring to authentication keys. |
| dashboard | portal, console | Use “dashboard” unless the product UI uses another term. |
| user | customer, client | Use “user” unless the context requires a different role. |
| delete | remove | Use “delete” for permanent actions. Use “remove” for reversible actions. |
