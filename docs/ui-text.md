# UI text rules

## Purpose

UI text should help users understand what an action does before they take it.

## Rules

- Use clear verbs.
- Keep button labels short.
- Use consistent terms.
- Avoid vague labels like **Submit** when a specific label is better.
- Match the user’s goal, not the system’s process.
