# Before and after examples

## Procedure example

### Before

To create a project, you will want to navigate to the dashboard, where you can click on the option for making a new project. After that, fill out the details and submit it.

### After

1. Open the dashboard.
2. Select **Create project**.
3. Enter a project name.
4. Select **Save**.

The new project appears in the project list.

## Error message example

### Before

Something went wrong.

### After

The file could not be uploaded because it is larger than 10 MB. Upload a smaller file and try again.
